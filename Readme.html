Creative and Beautiful WordPress Theme with a Large Header

Description

Raze is a unique, ultra creative multipurpose WordPress theme with complete support for WooCommerce. Raze Plus supports all basic features and is fully mobile friendly. And with Support for Custom Widgets, Slider and Featured Posts section, etc you can make this theme look even more beautiful on your website. Demo Here: http://demo.inkhive.com/raze-plus/
